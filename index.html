HOW DRISHTI GLOVES WORK

Four steps. One voice.

Each gesture is captured, classified, spoken, and connected — in under 400 milliseconds.

Capture

5 flex sensors detect finger bending angles. MPU6050 tracks wrist orientation and motion in 6 axes. ADS1115 converts analog signals to precise 16-bit digital data.

Classify

ESP32 packages sensor data and transmits over WiFi to a laptop running a trained scikit-learn Random Forest model. 40+ ISL signs are recognized with 92% accuracy.

Speak

The recognized ISL sign is instantly converted to spoken Hindi using pyttsx3 text-to-speech engine. The output is clear, natural-sounding speech that bridges the communication gap.

Connect

DRISHTI Gloves seamlessly connect to any WiFi network. Real-time data streaming means zero lag between gesture and speech. Future plans include Bluetooth LE for direct mobile pairing.

HOW KAI LOCKET WORKS

See. Listen. Think. Respond.

KAI processes the visual world and voice commands to deliver real-time spoken guidance for the visually impaired.

See

Pi Camera captures a wide-angle view of the world ahead. YOLOv8 runs object detection to identify people, furniture, vehicles, and obstacles in real-time.

Listen

INMP441 digital MEMS microphone captures voice commands. Vosk speech recognition processes Hindi commands entirely offline — no internet needed for core functions.

Think

Gemini API provides contextual intelligence for complex queries. The system combines object detection, currency recognition, and QR scanning to build a complete understanding of the scene.

Respond

pyttsx3 converts the AI's understanding into clear spoken Hindi. "Aapke saamne ek kursi hai" — KAI speaks naturally, giving blind users confidence to navigate independently.

How DRISHTI Works

01

Wear the Device

Slip on the Smart Gloves or hang the KAI Locket around your neck. Both devices power on with a single button press and connect automatically.

02

Sense the World

Flex sensors read finger positions for sign language. The Pi Camera captures the visual environment. INMP441 microphone listens for voice commands.

03

Process with AI

Machine learning models classify ISL gestures. YOLOv8 detects objects. Vosk transcribes speech. Gemini API answers complex questions. All in real-time.

04

Speak the Output

pyttsx3 text-to-speech converts the AI output into clear spoken Hindi. The gloves speak the recognized sign. KAI describes what it sees ahead.

05

Live Independently

With DRISHTI, deaf users can communicate freely and blind users can navigate confidently. Two systems, one mission: breaking barriers for millions of Indians.

How It All Connects

Technology Stack

HARDWARE

ESP32

ESP32

Dual-core WiFi/BLE microcontroller. Reads sensor data and transmits to ML model wirelessly.

ADS1115

ADS1115

16-bit ADC converter. Precisely digitizes analog flex sensor values for accurate gesture detection.

MPU6050

MPU6050

6-axis IMU with accelerometer and gyroscope. Tracks hand orientation and dynamic gestures.

Flex Sensors

Flex Sensors

5 resistive flex sensors per glove. Each maps finger bend angle to a resistance value for ISL detection.

Pi Zero 2WH

Pi Zero 2WH

Quad-core ARM brain of KAI Locket. Runs YOLOv8, Vosk, and Gemini API integration in a tiny form factor.

Pi Camera

Pi Camera

5MP camera module for KAI. Captures wide-angle frames for object detection and currency recognition.

INMP441

INMP441

Digital I2S MEMS microphone. Captures clear voice commands for Vosk speech recognition processing.

SPI Display

SPI Display

Small SPI-connected screen on KAI Locket. Shows system status, detected objects, and feedback information.

SOFTWARE & AI

Python

Python

Primary programming language for both systems. Handles ML inference, computer vision, and speech processing.

OpenCV

OpenCV

Computer vision library. Handles image preprocessing, QR code detection, and currency note scanning in KAI.

scikit-learn

scikit-learn

Random Forest classifier trained on ISL gesture data. Achieves 92% accuracy across 40+ signs with low latency.

YOLOv8

YOLOv8

Ultralytics object detection model. Identifies 80+ objects in real-time for KAI's navigation assistance.

Vosk

Vosk

Offline speech recognition for Hindi. Processes voice commands without internet — critical for rural accessibility.

pyttsx3

pyttsx3

Offline text-to-speech engine. Converts recognized signs and AI responses to spoken Hindi audio output.

Gemini API

Gemini API

Google's multimodal AI. Provides contextual intelligence for KAI — answering complex user questions about surroundings.

Three.js

Three.js

3D rendering for the DRISHTI website. Powers interactive glove and locket models with orbit controls.

GSAP

GSAP

GreenSock Animation Platform. Handles scroll-triggered animations, phase transitions, and smooth UI effects.
